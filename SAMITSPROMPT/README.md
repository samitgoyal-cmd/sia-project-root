# SamitsPrompt

## Purpose

This folder contains reusable prompt files for starting, closing, and note-capturing Codex sessions.

## Recommended Usage

You can start a session by saying:

`Please read and execute session_start_prompt in SAMITSPROMPT folder.`

You can close a session by saying:

`Please read and execute session_close_prompt in SAMITSPROMPT folder.`

You can ask for continuity notes by saying:

`Please read and execute capture_update_chat_notes_prompt in SAMITSPROMPT folder.`
